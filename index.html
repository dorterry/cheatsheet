Common CSS Properties

Name Description Example
background-attachment: sets whether a background image's position is fixed within the viewport, or scrolls with its containing block. background-attachment: scroll;
background-color: sets the color of the background background-color: green;
background-image: sets one or more background images on an element background-image: url("../../resournces/images/stones.png");
background-position: sets the initial position for each background image background-position: top;
background-repeat: sets how background images are repeated background-repeat: repeat;
