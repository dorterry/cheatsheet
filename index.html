Common CSS Properties

Name Description Example
background: sets all background style properties at once, such as color, image, origin and size, or repeat method background: green;
background-attachment: sets whether a background image's position is fixed within the viewport, or scrolls with its containing block. background-attachment: scroll;
background-color: sets the color of the background background-color: green;
background-image: sets one or more background images on an element background-image: url("../../resournces/images/stones.png");
background-position: sets the initial position for each background image background-position: top;
background-repeat: sets how background images are repeated background-repeat: repeat;
border: sets an element's border border: solid red;
border-bottom: sets an element's bottom border border-bottom: dashed red;
border-bottom-color: sets the color of an element's bottom border border-bottom-color: red;
border-bottom-style: sets the line style of an element's bottom border border-bottom-style: dotted;
border-bottom-width: sets the width of the bottom border of an element border-bottom-width: 4px;
border-color: sets the color of an element's border border-color: red;
border-left: sets all the properties of an element's left border border-left: dashed red;
border-left-color: sets the color of an element's left border border-left-color: red;
border-left-style: sets the line style of an element's left border border-left-style: dotted;
border-left-width: sets the width of the left border of an element border-left-width: thick;
border-right: sets all the properties of an element's right border border-right: dashed red;
border-right-color: sets the color of an element's right border border-right-color: red;
border-right-style: sets the line style of an element's right border border-right-style: dashed;
border-right-width: sets the width of the right border of an element border-right-width: thick;
border-style: sets the line style for all four sides of an element's border border-style: dotted;
border-top: sets all the properties of an element's top border border-top: solid;
border-type-color: property sets the color of an element's top border border-top-color: red;
border-type-style: sets the line style of an element's top border border-top-style: dashed;
border-top-width: sets the width of the top border of an element border-top-width: 4px;
border-type-style: sets the line style of an element's top border border-top-style: dashed;
border-width: sets the width of an element's border border-width: 0 4px 8px 12px;
clear: sets whether an element must be moved below (cleared) floating elements that precede it - applies to floating and non-floating elements clear: left;
clip: defines a visible portion of an element - applies only to absolutely positioned elements clip: auto; clip: rect(1px, 10em, 3rem, 2ch);
color: sets the foreground color value of an element's text color: #00ff00;
cursor: sets the mouse cursor, if any, to show when the mouse pointer is over an element cursor: help;
display: sets an element's inner and outer display types display: inline-block;
filter: applies graphical effects like blur or color shift to an element filter: blur(5px);
float: places an element on the left or right side of its container, allowing text and inline elements to wrap around it float: left;
font: sets all the different properties of an element's font font: italic 1.2em "Fira Sans", serif;
font-family: specifies a prioritized list of one or more font family names and/or generic family names for the selected element font-family: Georgia, serif;
font-size: sets the size of the font font-size: 12px;
font-family: allows you to set all the font variants for a font font-variant: normal;
